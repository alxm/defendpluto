Defend Pluto
============

Defend Pluto is a vertical space shooter for the Gamebuino META. This is Beta 1 with 5 levels, more coming up. Visit https://www.alxm.org/games/gamebuino.html for updates and my other games.

Install
-------

Copy the `DefendPluto` folder to your SD card.

Controls
--------

* Move with the dpad
* Shoot with the A button
* Pause with the Menu button
